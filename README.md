# ExtraTreesClassifier_LinearSVC

Задача бинарной классификации для предсказания решения о покупке недвижимости на основе характеристик объекта, финансовых показателей клиента и параметров локации. Реализована с использованием двух алгоритмов машинного обучения (Extremely Randomized Trees и Linear Support Vector Classification) и методов анализа и обработки данных.

Оценка качества моделей:
Основная метрика: F1-Score (сбалансированная мера)
Дополнительные метрики: Precision, Recall, Accuracy

Технологии:
Анализ данных: pandas, numpy, matplotlib, seaborn
Машинное обучение: scikit-learn, phik
Предобработка: scikit-learn preprocessing
Визуализация: matplotlib, seaborn

Установка и запуск:
1. Убедитесь, что у вас установлен Python 3.8 или выше и необходимые библиотеки, при необходимости выполните:
pip install pandas numpy matplotlib seaborn scikit-learn phik
2.Поместите файл датасета global_house_purchase_dataset.csv в корневую директорию и запустите файл с кодом
